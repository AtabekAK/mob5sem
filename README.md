

# Мобильное приложение "Трекер Привычек" (Habit Tracker Game)

Проект выполнен студентом группы 231-322 **Абдыкайымовым Атабеком** в рамках курса "Мобильная разработка".

## О проекте

"Трекер Привычек" — это полнофункциональное мобильное приложение, разработанное на **React Native** с использованием **Expo**. 
Цель приложения — помочь пользователям формировать полезные привычки через интерактивный игровой процесс. Приложение работает полностью **офлайн**, храня все данные локально на устройстве пользователя.

Каждая привычка представлена как игровое задание, выполнение которого отслеживается, награждается очками и влияет на общий прогресс, превращая рутину в увлекательную игру.

## Основной функционал

### 1. Аутентификация и профиль пользователя
*   **Регистрация и Вход:** Реализована локальная система аутентификации. Все данные пользователя хранятся на устройстве в `AsyncStorage`.
*   **Профиль пользователя:** На экране "Настройки" отображается профиль пользователя, включая логин, дату регистрации и возможность загрузить/изменить **аватарку** из галереи устройства (с кадрированием 1:1).
*   **Управление сессией:** Приложение запоминает вошедшего пользователя и предоставляет функцию "Выход из аккаунта".

### 2. Управление привычками (CRUD)
*   **Создание и Редактирование:** Пользователь может создавать и редактировать привычки через интуитивное модальное окно, настраивая:
    *   Название
    *   Длительность выполнения (в минутах)
    *   Дни недели
    *   Иконку (из предложенного набора SVG)
    *   Цвет темы
*   **Ограничение:** Для повышения концентрации действует ограничение — не более 5 привычек на один день.
*   **Удаление:** Безопасное удаление привычек через модальное окно подтверждения.

### 3. Геймплей и интерактивность
*   **Таймер выполнения:** Каждая привычка на сегодняшний день имеет встроенный таймер с возможностью запуска и паузы. Успешное завершение таймера автоматически отмечает привычку как выполненную и начисляет **10 очков**.
*   **Ручное выполнение:** Возможность мгновенно отметить привычку выполненной без таймера. За это начисляется **5 очков**.
*   **Система очков и стриков:** Приложение отслеживает:
    *   Общее количество очков.
    *   **Стрики (серии)** для каждой привычки — количество последовательных дней выполнения.
*   **Блокировка действий:** Выполнять привычки (запускать таймер, отмечать вручную) можно **только для текущего дня**. Привычки на другие дни отображаются, но неактивны.

### 4. Статистика и Прогресс
*   **Прогресс-бар дня:** На главном экране отображается наглядный прогресс-бар выполнения привычек на сегодня.
*   **Общая статистика:** На экране "Прогресс" собрана общая игровая статистика: общее число выполненных привычек, лучший стрик среди всех привычек и суммарное количество очков.
*   **Детальная статистика по привычке:** Для каждой привычки доступна подробная аналитика, включая текущий и лучший стрик, а также **календарь выполнений**, визуализирующий регулярность.

## Используемые технологии и библиотеки

*   **Платформа:** React Native с Expo
*   **Язык:** TypeScript
*   **Навигация:**
    *   `@react-navigation/native`
    *   `@react-navigation/bottom-tabs` (для основной навигации)
    *   `@react-navigation/stack` (для экранов входа и регистрации)
*   **Хранение данных:** `@react-native-async-storage/async-storage` (локальная база данных)
*   **Работа с мультимедиа:**
    *   `expo-image-picker` (для загрузки аватара из галереи)
    *   `react-native-svg` (для отображения кастомных иконок)
*   **Управление состоянием:** Встроенные React Hooks (`useState`, `useEffect`, `useMemo`, `useCallback`).

## Инструкции по запуску

### 1. Клонирование репозитория
```bash
git clone [https://github.com/AtabekAK/mob5sem.git]
cd [habit-tracker-game]
```

### 2. Установка зависимостей
```bash
npm install
```

### 3. Запуск проекта
Для запуска на мобильном устройстве через **Expo Go**:
```bash
npx expo start
```
После этого отсканируйте QR-код с помощью приложения Expo Go (Android) или камеры (iOS).

Для запуска в **веб-браузере**:
```bash
npx expo start --web
# или нажмите 'w' в меню терминала
```
